#  Moodle Client Mobile

Aplicación móvil desarrollada en Flutter que permite a los estudiantes acceder a la plataforma Moodle mediante servicios web REST, facilitando la consulta de cursos, actividades, foros y tareas desde dispositivos móviles.

---

##  Objetivo del Proyecto

Desarrollar un cliente móvil que se conecte a Moodle utilizando API REST, permitiendo gestionar información académica de forma remota.

---

##  Funcionalidades Implementadas

###  Autenticación
- Inicio de sesión mediante usuario y contraseña
- Generación y uso de token Moodle
- Validación de credenciales

---

### Gestión de Cursos
- Visualización de cursos matriculados
- Navegación hacia detalle del curso
- Interfaz moderna tipo Canva

---

###  Actividades del Curso
- Listado de actividades disponibles
- Visualización de información académica

---

###  Foros
- Visualización de discusiones
- Publicación de mensajes

---

###  Sesión de Usuario
- Botón de cerrar sesión
- Eliminación de navegación anterior
- Seguridad básica de acceso
---

##  Arquitectura del Proyecto

La aplicación fue desarrollada usando arquitectura por capas
